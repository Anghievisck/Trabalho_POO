# 👾 Space Invaders em Java (Trabalho_POO)

Um clone do clássico jogo **Space Invaders** desenvolvido inteiramente em **Java**. Este projeto foi construído para aplicar e consolidar na prática os pilares da Programação Orientada a Objetos (POO), lidando com herança, polimorfismo, encapsulamento e gerenciamento de estados.

## 🚀 Visão Geral e Funcionalidades
*(Sinta-se livre para editar ou adicionar as mecânicas que você implementou)*

- Movimentação fluida da nave controlada pelo jogador.
- Sistema de disparo de projéteis.
- Lógica de movimentação da horda inimiga e detecção de colisões.
- Gerenciamento de recursos visuais (sprites) e interface do jogo.

## 🛠️ Tecnologias Utilizadas
- **Java** (Linguagem principal, 99% do código)
- **Makefile** (Para automação da compilação e execução)

## 📁 Estrutura do Projeto
A arquitetura do projeto foi dividida da seguinte forma:

- 📂 `Game/`: Contém todo o código-fonte `.java`. Aqui estão as classes que definem a lógica do jogo, a janela, os atores (nave, inimigos, tiros) e a renderização principal.
- 📂 `Resources/`: Diretório dedicado a armazenar os recursos de mídia do jogo, como imagens, sprites dos invasores e possíveis efeitos sonoros.
- 📄 `makefile`: Script configurado para facilitar o processo de compilação do código Java sem precisar rodar comandos complexos no terminal.
